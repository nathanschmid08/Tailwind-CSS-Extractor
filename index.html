Tailwind CSS Extractor

Extract all used Tailwind CSS classes from a page and generate your own custom stylesheet. Clean. Secure. Offline.

About

Stop relying on Tailwind CDNs. This tool scans a webpage, extracts only the used Tailwind classes, and gives you a clean, lightweight stylesheet to host locally. Perfect for projects where performance and privacy matter.

Tutorial

Read the quick-start guide and get going in under 30 seconds. It covers everything from installation to style generation.


Quick-start guide

  1. Make sure the extension is active
  2. Open your tailwind website
  3. Click on the puzzle icon on the top right corner
  4. A popup will open, click on the button
  5. Done! You generated the CSS

Installation Guide

Note

This extension works best with Google Chrome.

  1. Download the extension (see below)
  2. Open Google Chrome
  3. Go to chrome://extensions
  4. Enable Developer Mode
  5. Click “Load unpacked”
  6. Select the downloaded folder
  7. Done! Extension is now active
